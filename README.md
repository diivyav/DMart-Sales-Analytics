 DMart Retail Sales Analytics Dashboard

 Project Overview

The **DMart Retail Sales Analytics Dashboard** is an Excel-based data analytics project designed to analyze supermarket sales performance and customer purchasing behavior. The dashboard transforms raw sales data into meaningful business insights using Excel's analytical and visualization capabilities.

The project focuses on identifying top-performing products, branch performance, customer satisfaction levels, and sales trends across different time periods. Interactive charts, PivotTables, and formulas were used to create a comprehensive dashboard that supports data-driven decision-making.



Tools Used

* Microsoft Excel
* PivotTables
* Pivot Charts
* VLOOKUP
* SUMIF
* Conditional Formatting
* Data Cleaning & Transformation
* Dashboard Design



 Key Insights

 Product Performance

Food and Beverages emerged as the top-selling product category, generating the highest sales volume among all product lines.

 Branch Performance

The **Giza Branch** recorded the highest revenue, making it the best-performing branch in terms of sales generation.

 Customer Satisfaction

 The average customer rating across all transactions was **6.97**, indicating a moderate level of customer satisfaction.

 Sales Trends

 The majority of sales occurred during the **Evening** period, suggesting peak customer activity during later hours of the day.
